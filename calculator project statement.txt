Problem Statement

The goal of this project is to build a functional calculator using Python that can perform essential arithmetic operations such as addition, subtraction, multiplication, and division. The calculator should provide accurate results, handle invalid inputs gracefully, and serve as a foundational project for understanding Python programming logic and user interaction.

Scope of the Project

Implement a fully operational command-line-based calculator.

Support basic mathematical operations.

Provide input validation to avoid errors during execution.

Display user-friendly messages and results.

Serve as a building block for future enhancements such as GUI, scientific functions, or advanced mathematical features.

Target Users

Students learning Python programming.

Beginners practicing basic mathematical logic in coding.

Developers looking for a simple and extendable calculator template.

Anyone needing a quick and lightweight tool for arithmetic calculations.

High-Level Features

User-friendly input system.

Support for addition, subtraction, multiplication, and division.

Error handling for invalid inputs or operations (e.g., division by zero).

Clean and readable program structure.

Easily extendable for more advanced features in the future.
